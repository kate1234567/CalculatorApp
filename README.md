# Calculator App

## Описание

Calculator App — простое Android-приложение калькулятора, разработанное на Kotlin.

Приложение выполняет базовые арифметические операции и корректно работает при смене ориентации экрана.

Проект выполнен в рамках лабораторной работы по дисциплине «Мобильная разработка».

## Используемые технологии

- Kotlin
- Android SDK
- Android Studio
- Gradle

## Основные возможности

- выполнение арифметических операций;
- вычисления в типе `Double`;
- собственная экранная клавиатура;
- сохранение состояния при повороте экрана;
- отсутствие блокировки поворота экрана.

## Запуск проекта

1. Открыть проект в Android Studio.
2. Дождаться синхронизации Gradle.
3. Запустить приложение на эмуляторе или Android-устройстве.
